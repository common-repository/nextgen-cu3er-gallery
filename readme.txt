== Description ==

Nextgen Cu3er  Gallery: A Wordpress Plugin that allows you to use the amazing [Cu3er Gallery](http://www.progressivered.com/cu3er/) viewer on your NextGen-Gallery(http://wordpress.org/extend/plugins/nextgen-gallery/) galleries.

In order to use the cu3er galery viewer on your blog, you need to be using the best Wordpress gallery manager: [NextGen Gallery](http://wordpress.org/extend/plugins/nextgen-gallery/).

Also in order to link images you need to use [Nextgen Custom Fields Plugin](http://wordpress.org/extend/plugins/nextgen-gallery-custom-fields/)

 = wp_head() is need on your header template! =

 = Main features: = 

    * Multiple Widgets Allowed
	* Allow Posts Shortcode
	* Random Effects
	* Allow Title, Description and URL on desired image
	* Works only with a NextGen Gallery

 = Admin page options: = 

	* Widget: width, height and NextGen Gallery
	* Post Format: [cu3er id="NextGEN gallery ID" width="100" height="200" align="center"]

== Installation ==

1. Upload nextgen-cu3er-gallery folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create You Gallery with NextGEN
1. (Optional) Fill out the Title and Description image fields
1. Go to Widgets and activate NextGEN Cu3er Gallery
1. Go to Posts and use [cu3er id="NextGEN gallery ID" width="100" height="200" align="center"]

(Optional) If desired to link images

1. Create a Image Custom Field named "url" (Field Type = Text Imput)
1. Fill out this field on the images you want to link
